# 📊 Retail Sales Analytics Dashboard

## 📌 Overview
This project demonstrates an end-to-end retail sales analytics solution built using SQL Server and Power BI. The dashboard provides interactive visualizations and key business insights to help analyze sales performance.

## 🚀 Technologies Used
- SQL Server
- Power BI
- DAX
- Excel

## 📂 Project Files
- Retail Sales Analytics Dashboard.pbix
- SQL_Queries.sql
- Sample - Superstore.csv
- Dashboard Screenshot

## 📈 Dashboard Features
- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Profit Margin
- Sales by Category
- Sales Trend
- Regional Analysis
- Interactive Slicers
